## Mac Mic Widget v0.1.7

This release improves quick-glance readability of microphone state in the menu bar, with a clearer muted signal and stable icon footprint during toggles.

### Highlights
- Muted state now uses a stronger dedicated symbol (`mic.slash.fill`) plus subtle red tint for faster visual recognition.
- Active state keeps variable signal-meter rendering with improved low-level visibility mapping.
- Status item width remains stable (`NSStatusItem.squareLength`) to avoid visible layout jumps on mute/unmute.
- Added unit tests for status icon presentation logic (muted symbol, clamping, low-level boost behavior).

### Requirements
- macOS 14+
- Apple Silicon (arm64) build

### Known limitations
- Some USB/Bluetooth headsets expose input gain in non-standard ways and may not support write via CoreAudio scalar property.
- App reads/writes the system default input device only.
