## Mac Mic Widget v0.1.0

First public release of Mac Mic Widget - a lightweight macOS menu bar app for quick microphone mute/unmute and volume visibility.

### Highlights
- Menu bar microphone icon with visual level state.
- Fast toggle between mute (`0%`) and last non-zero microphone level.
- Popover UI with level percent, badge, progress bar, mute/unmute button, and quit action.
- Left click opens/closes popover, right click toggles mute/unmute without opening popover.

### Requirements
- macOS 14+
- Apple Silicon (arm64) build.

### Known limitations
- Some USB/Bluetooth headsets expose input gain in non-standard ways and may not support write via CoreAudio scalar property.
- App reads/writes the system default input device only.
- On this first release, build is unsigned/not notarized. On first launch, users may need to allow app execution in `Privacy & Security`.
