## Mac Mic Widget v0.1.6

This release improves popover readability and action clarity with refined typography, spacing, and state-driven mute button behavior.

### Highlights
- Refined popover visual hierarchy with clearer section rhythm and stronger primary value emphasis.
- Improved secondary controls layout for global hotkey management with balanced button sizing.
- Updated mute action button to centered label, shorter `Mute`/`Unmute` text, and state-based color (`Mute` in red, `Unmute` in accent color).
- Increased popover size to better accommodate the updated spacing and typography.

### Requirements
- macOS 14+
- Apple Silicon (arm64) build

### Known limitations
- Some USB/Bluetooth headsets expose input gain in non-standard ways and may not support write via CoreAudio scalar property.
- App reads/writes the system default input device only.
