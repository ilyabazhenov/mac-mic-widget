## Mac Mic Widget v0.1.5

This release improves popover volume UX by unifying display and control into one slider and removing visible jitter during and after drag.

### Highlights
- Popover now uses a single `Input level` slider as both current-level indicator and adjustment control.
- Fixed slider visual bounce when ending drag by stabilizing commit order in the SwiftUI view.
- Smoothed post-write state updates in microphone service to ignore transient backend values until volume settles.
- Added regression tests for optimistic slider updates and transient read suppression.

### Requirements
- macOS 14+
- Apple Silicon (arm64) build

### Known limitations
- Some USB/Bluetooth headsets expose input gain in non-standard ways and may not support write via CoreAudio scalar property.
- App reads/writes the system default input device only.
