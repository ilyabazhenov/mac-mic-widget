## Mac Mic Widget v0.1.8

This release updates the out-of-the-box global hotkey and aligns docs/tests with the new default behavior.

### Highlights
- Default global hotkey is now `Shift + Option + Z` (`⌥⇧Z`).
- Added a regression test to keep default hotkey display behavior stable.
- Updated README behavior section to reflect the new default shortcut.

### Requirements
- macOS 14+
- Apple Silicon (arm64) build

### Known limitations
- Some USB/Bluetooth headsets expose input gain in non-standard ways and may not support write via CoreAudio scalar property.
- App reads/writes the system default input device only.
