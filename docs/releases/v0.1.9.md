## Mac Mic Widget v0.1.9

This release focuses on release quality and open-source readiness by improving CI reliability and repository documentation.

### Highlights
- Added a GitHub Actions CI workflow for tests and coverage on pushes and pull requests.
- Updated CI Swift setup to an available toolchain version and removed `ripgrep` as a coverage parser dependency.
- Added `LICENSE`, `SECURITY.md`, and `CONTRIBUTING.md` to make project usage and contribution expectations explicit.

### Requirements
- macOS 14+
- Apple Silicon (arm64) build

### Known limitations
- Some USB/Bluetooth headsets expose input gain in non-standard ways and may not support write via CoreAudio scalar property.
- App reads/writes the system default input device only.
