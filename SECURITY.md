# Security policy

## Supported versions

Security fixes are considered for the latest release and the `main` branch.

## Reporting a vulnerability

Please report security issues **privately** via [GitHub Security Advisories](https://github.com/ilyabazhenov/mac-mic-widget/security/advisories/new) for this repository (preferred).

Do not open a public issue for undisclosed vulnerabilities.

## Scope

Mac Mic Widget is a local menu bar utility. It does not run a network server or collect telemetry. Reports are still welcome for issues that could affect users through malicious distribution, local privilege boundaries, or unsafe defaults in the shipped app.

## What to expect

We aim to acknowledge reports within a few business days. If you need encrypted email instead of GitHub, say so in a minimal public issue (no exploit details) and we can coordinate out of band.
