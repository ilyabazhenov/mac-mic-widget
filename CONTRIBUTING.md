# Contributing

Спасибо за интерес к проекту. Ниже — минимум, чтобы правки были предсказуемыми и быстрее принимались.

## Перед изменениями

1. Прочитай [AGENTS.md](AGENTS.md) — там контракты UI, аудио и чеклист ручной проверки.
2. Для поведения продукта смотри [docs/PRODUCT_REQUIREMENTS.md](docs/PRODUCT_REQUIREMENTS.md).

## Сборка и тесты

```bash
make run    # локальный запуск
make test   # обязательно перед PR
make coverage   # опционально, порог настраивается COVERAGE_MIN
```

## Что не коммитить

- Каталоги и артефакты из [.gitignore](.gitignore): например `/dist`, `/coverage`, `.build`, ключи и `.netrc`.
- Секреты Apple Developer, пароли notarization и т.п. — только локально или в CI-секретах, не в репозитории.

Лицензия проекта: [LICENSE](LICENSE).

## Pull requests

- Один PR — одна тема (фича или исправление).
- Краткое описание «что» и «зачем»; если менялось поведение — как проверил (в т.ч. пункты из AGENTS.md).

Код и комментарии в исходниках — на английском, как в остальном репозитории.
